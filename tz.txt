Необходимо реализовать виджет (компонент, плагин), который отрисует N последних сообщений из заданного фида в указанный HTML контейнер. Виджет должен обновлять содержимое списка с заданным интервалом, добавляя новые сообщения и удаляя старые. Желательно, чтобы добавлялись и удалялись только необходимые элементы, а не перерисовывался весь список полностью.'

В качество параметров виджет должен получать следующие аргументы: 
	1. Ссылка на фид в JSON формате 
	2. Кол-во последних сообщений, которые нужно отрисовать 
	3. Интервал, с которым данные нужно апдейтить 

В качестве тестового фида используйте http://api.massrelevance.com/reccosxof/matchtrax_hashclash_featured_tweets.json. Данный фид также поддерживает JSONP. Документацию по нему можно найти здесь: http://dev.massrelevance.com/docs/api/v1.0/stream/#ref-params-standard. Достаточно выводить в список имя автора сообщения и текст сообщения.
